# ImmoSketch

ImmoSketch is een eenvoudige en efficiënte 2D-grondplan tekenapp, ontwikkeld voor energiedeskundigen en EPB-verslaggevers.

## ✏️ Functionaliteiten
- Grondplannen tekenen op schaal
- Werken met muren, ruimtes, ramen en deuren
- Snapping voor nauwkeurig tekenen
- Gebruik op tablet en smartphone
- Voorbereid voor export naar SketchUp

## 🎯 Doel
Snelle opmaak van grondplannen tijdens plaatsbezoeken voor o.a. EPC-certificaten.

## 🚀 Technologie
- HTML
- CSS
- JavaScript
- LocalStorage
